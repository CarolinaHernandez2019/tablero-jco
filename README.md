# Tablero JCO - Priorización Territorial

Tablero interactivo para la priorización territorial del programa **Jóvenes con Oportunidades (JCO)** de la Secretaría de Integración Social de Bogotá.

## Características

- **Mapa interactivo** de Bogotá por UPZ con datos de población juvenil vulnerable
- **Filtros** por localidad y rango de priorización
- **Visualización** de límites de localidades
- **Análisis** por categorías SISBEN (A, B, C, D)
- **Desglose** por género (hombres/mujeres)
- **Descarga** de datos filtrados

## Datos

- **Fuente:** Base de datos SISBEN IV
- **Población objetivo:** Jóvenes entre 14 y 28 años
- **Grupos vulnerables:** A (pobreza extrema), B (pobreza moderada), C (vulnerable)

## Uso

El tablero permite:
1. Explorar el mapa de priorización por UPZ
2. Filtrar por localidad específica
3. Ver estadísticas por categoría SISBEN
4. Analizar distribución por género
5. Descargar datos para análisis adicional
